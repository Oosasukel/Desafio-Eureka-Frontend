# Tech Stack

**Client:**

- React.js
- Next.js
- Styled-Components
- TypeScript

**Server:**

- Node.js
- Express
- Redis

# Run Locally

### Clone the project

```bash
  git clone https://github.com/Oosasukel/Desafio-D1-Frontend
```

### Go to the project directory

```bash
  cd Desafio-Eureka-Frontend
```

### Install dependencies

```bash
  yarn
```

### Start the server

```bash
  yarn dev
```

# Run Tests

```bash
  yarn test
```
